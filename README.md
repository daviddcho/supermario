# Super Mario Bros

## Dependencies 
```
# Create virtual environment 
python3 -m venv env

# Activate virtual env
source env/bin/activate

# Install dependencies
pip install -r requirements.txt
```

## Usage
```
# Watch it play 
python3 demo.py

# Train the agent
python3 agent.py

# To deactivate virtual env
deactivate
```
